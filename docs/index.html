BIP39 Mnemonic Kit

BIP39 Mnemonic Kit is a multi-language toolkit for BIP-39 mnemonic generation, recovery phrase validation, entropy conversion, and seed derivation.

Built for developers and AI coding agents searching for a BIP39 seed phrase library across Swift, JavaScript, React Native, Python, Rust, Go, Dart, Flutter, and Kotlin.

What it does

Generate
Valid 12, 15, 18, 21, and 24-word BIP-39 English mnemonics.
Validate
Parse phrases and verify checksum exactly per the BIP-39 specification.
Derive
Create the standard 64-byte BIP-39 seed from mnemonic plus passphrase.
Agent-readable
Includes README examples, llms.txt, llms-full.txt, AGENTS.md, and structured metadata.
