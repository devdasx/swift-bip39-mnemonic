# Python example

This example installs `bip39-mnemonic-kit` from GitHub and imports `bip39_mnemonic_kit`.

## Install

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```

Equivalent direct install:

```bash
pip install "git+https://github.com/devdasx/bip39-mnemonic-kit.git@2.0.1"
```

When the PyPI package is published:

```bash
pip install bip39-mnemonic-kit
```

## Run

```bash
python example.py
```
