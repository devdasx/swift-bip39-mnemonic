SwiftBIP39

SwiftBIP39 is a Swift Package Manager library for BIP-39 mnemonic generation, recovery phrase validation, and seed derivation on Apple platforms.

Built for developers searching for a Swift BIP39 library, mnemonic generator, seed phrase package, or wallet recovery phrase toolkit.

View on GitHub Latest Release

What it does

Generate
Valid 12, 15, 18, 21, and 24-word BIP-39 English mnemonics.
Validate
Parse phrases and verify the checksum exactly per the BIP-39 specification.
Derive
Create the standard 64-byte BIP-39 seed from mnemonic + passphrase.
Ship
Integrate through Swift Package Manager from GitHub as the source of truth.

Install with Swift Package Manager

dependencies: [
    .package(url: "https://github.com/devdasx/swift-bip39-mnemonic.git", from: "1.0.1")
]

Usage

import SwiftBIP39

let mnemonic = try BIP39.generate(strength: .bits128)
let phrase = mnemonic.phrase

let parsed = try BIP39.parse(phrase)
let isValid = BIP39.validate(phrase)
let seed = BIP39.seed(from: parsed, passphrase: "optional-passphrase")
